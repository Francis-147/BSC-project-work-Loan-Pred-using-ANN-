# BSC-project-work-Loan-Pred-using-ANN-and-Visual-Analytics

This project aims to develop a data driven loan management system that integrates structured database queries with visual analytics and predictive modeling that will study the pattern of past data using neural network layers. The system will use SQL to retrieve relevant loan and customer data, Tableau to display meaningful insights from past loan records, and a credit risk evaluation model to help determine whether a loan request should be approved or rejected.
